## Getting Started

### Prerequisites

- [Snakemake](https://snakemake.readthedocs.io/)
- [R](https://www.r-project.org/)
- Any additional R packages used in the scripts

### Installation

Clone the repository:
```sh
git clone https://github.com/bailey-lab/Repo_Template.git
cd Repo_Template
```
